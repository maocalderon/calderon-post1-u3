## Descripción del Proyecto
Este proyecto consiste en una página de perfil profesional estática desarrollada con HTML5 semántico y CSS3. Se centra en la implementación del **Box Model**, posicionamiento avanzado (`fixed`, `absolute`, `relative`) y el uso de **Custom Properties** para una gestión eficiente de estilos.

## Características Técnicas
- [cite_start]**Box Model:** Uso de `box-sizing: border-box` en todos los elementos[cite: 775, 780].
- [cite_start]**Posicionamiento:** Header fijo en la parte superior y badge de habilidades posicionado de forma absoluta sobre el avatar[cite: 781, 978, 1052].
- [cite_start]**Selectores:** Uso de selectores avanzados, combinadores de hijo directo y pseudo-clases como `:hover` y `:focus`[cite: 779, 1134, 1225].
- [cite_start]**Accesibilidad:** Formulario de contacto con indicadores de foco (focus ring) claros para navegación por teclado[cite: 1225, 1278].
